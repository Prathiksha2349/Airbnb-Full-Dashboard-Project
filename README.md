## 📝 Project Overview
This project focuses on analyzing Seattle Airbnb listing data to understand market dynamics, pricing trends, and revenue potential for hosts. The project follows the structured data analyst bootcamp curriculum by Alex the Analyst.

### Key Business Questions Answered:
* **Pricing Dynamics**: What is the average price per bedroom across different Seattle zip codes?
* **Revenue Analysis**: Which calendar weeks and months generate the highest revenue peaks for property owners?
* **Inventory Overview**: What types of properties and bedroom counts dominate the local market?

## 🛠️ Tools & Technologies Used
* **Tableau Desktop / Public**: For advanced calculations, data visualizations, and building the final interactive dashboard.
* **Microsoft Excel**: For data preparation, structuring, and consolidating the three separate source files (Listings, Calendar, and Reviews) into a unified dataset.

## 💾 Dataset Reference
The data utilized for this analysis consists of extensive Seattle Airbnb listings, covering room details, locations, and historical booking calendar trends. Three separate datasets were structured together into a single Excel file to build this project.
* *Note: The raw datasets are originally sourced from [Alex The Analyst's Kaggle Profile](https://www.kaggle.com/datasets/airbnb/seattle).*
